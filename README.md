#  Band Name Generator

This is a simple Python project that generates random band names based on user inputs.

##  What It Does
- Prompts for two inputs (like your city & pet name)  
- Generates a unique, catchy band name  
- Runs entirely in the console

##  Tech Stack
- Python 3.x

##  Run it yourself
    git clone https://github.com/EmreDursun2712/band-name-generator.git
    cd band-name-generator
    python3 app.py

##  Example
- What's the name of the city you grew up in? London
- What's your pet's name? Tiger
- Your band name could be: London Tiger
